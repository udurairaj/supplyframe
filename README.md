# Movie Madness!

Movie Madness is a local web app that displays movie information pulled from The Movie DB API. The homepage displays movies in theaters and a search bar which can be used to search and display movies. The web app is built with Express with EJS, Bootstrap, vanilla JS, GitHub, and Nightwatch for basic testing.

## Installation

Clone into this git repository and enter into the root directory.

Install [Node.js](https://nodejs.org/en/) and the package manager [npm](https://www.npmjs.com/get-npm) to install the dependencies necessary.

```bash
npm install
```

## Usage

To start the web app, run

```bash
npm start
```

and visit "http://localhost:8080".

To run the basic Nightwatch tests, then run

```bash
npm test
```
